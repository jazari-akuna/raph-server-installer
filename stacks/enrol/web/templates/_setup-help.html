set up your device

Pick your operating system, install AmneziaWG, then import the {{.PeerName}}.conf shown above (or scan the QR on a phone). The same .conf works on every platform.

Windows
  1. Download the AmneziaWG client for Windows from amnezia.org/downloads and install the .msi.
  2. Open AmneziaWG and click Import config from file.
  3. Select {{.PeerName}}.conf (download it from this page).
  4. Click connect.
macOS
  1. Download the AmneziaWG client for macOS from amnezia.org/downloads (Apple Silicon and Intel are both on the same page).
  2. Drag the app to Applications and open it.
  3. Click Import config from file and select {{.PeerName}}.conf.
  4. Click connect.
Linux (CLI)
  1. Install amneziawg-tools:
    • Debian/Ubuntu: 
      sudo apt install amneziawg-tools amneziawg-dkms
    • Fedora: see amneziawg-linux-kernel-module.
  2. Place the .conf in the right directory: 
    sudo install -m600 {{.PeerName}}.conf /etc/amnezia/amneziawg/{{.PeerName}}.conf
  3. Bring the tunnel up: 
    sudo awg-quick up {{.PeerName}}
  4. To stop: 
    sudo awg-quick down {{.PeerName}}
Linux (GUI)
  1. Alternative to the CLI: install the AmneziaVPN AppImage from amnezia.org/downloads.
  2. Open the app and click Import config from file.
  3. Select {{.PeerName}}.conf and connect.
Android
  1. Install AmneziaWG from Google Play or F-Droid.
  2. In the app, tap +.
  3. Tap Scan QR code (recommended) and scan the QR below, or tap Import from file and select {{.PeerName}}.conf.
