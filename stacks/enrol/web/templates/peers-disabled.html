AmneziaWG (the regional split-tunnel VPN that backs the devices feature) is not enabled on this server. AmneziaWG is opt-in: a default install skips it because most operators don't need a per-user split tunnel.

{{if .ViewerIsAdmin}}

To enable it, run scripts/install-gw0.sh on the host (or set SKIP_GW0=0 before re-running scripts/bootstrap-phase2.sh). Once gw0 is up, this page will let every user enrol their own devices.

{{else}}

Ask the server admin to enable AmneziaWG (the install script requires root on the host). Once it's up, you'll be able to enrol your own devices from this page.

{{end}}
