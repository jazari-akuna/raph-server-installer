devices

AmneziaWG (the regional split-tunnel VPN that backs the devices feature) is not enabled on this server. AmneziaWG is opt-in: a default install skips it because most operators don't need a per-user split tunnel.

To enable it, run scripts/install-gw0.sh on the host (or set SKIP_GW0=0 before re-running scripts/bootstrap-phase2.sh). Once gw0 is up, this page will list peer devices grouped by user and let you add new ones.

No peers exist yet — there is nothing to migrate or back up before enabling. Existing user accounts and storage volumes are unaffected.
