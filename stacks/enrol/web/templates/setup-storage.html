storage

Nextcloud stores each user's files under /srv/store/cloud-data/<user>/files on the host disk. Pick a default per-user quota; Nextcloud enforces it on writes and the dashboard's storage panel renders it as the per-user nominal column.

Disk underlying /srv/store: {{prettyBytes .DiskFreeBytes}} free of {{prettyBytes .DiskTotalBytes}} total. The quota is a soft cap on each user's files; setting it larger than the disk is allowed (and harmless until users actually write that much).
