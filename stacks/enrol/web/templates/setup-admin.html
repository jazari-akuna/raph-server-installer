first admin

This is the operator account. It gets the admins group in Authelia (which is required to reach the enrol UI and Portainer) and a host-side OS user. The login password is also used as the LUKS passphrase for the per-user encrypted volume; the login interceptor auto-unlocks it on every successful sign-in.

Add additional users from the enrol UI after setup completes. There is intentionally no "skip" here — an installer with no admin account is unrecoverable without SSH.

two-factor (TOTP)

Default is off. The wizard does NOT require TOTP at first login because a misconfigured authenticator app is a common way to lock yourself out of a fresh install. Once you can sign in normally, enable TOTP from the user detail page.
