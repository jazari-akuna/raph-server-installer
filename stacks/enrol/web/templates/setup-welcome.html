welcome

This wizard finishes the install for the host you just bootstrapped. It runs in your browser, gated by the one-shot token printed at the end of bootstrap. You will not need to SSH to the server.

There are six steps. Three collect input (domain confirmation, DNS provider credentials, first admin account). The fourth runs the configuration end-to-end: writing the admin user, issuing a wildcard TLS certificate, and wiring the reverse proxy. Cert issuance can take one to three minutes; the page will stream progress as it happens.

After this finishes, the wizard locks itself permanently. Re-running it requires deleting /srv/store/.setup-complete on the host as root.
